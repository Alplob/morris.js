Installation

You can either download the latest version of Morris.js06 from the GitHub releases.

Or install via npm:

            
    npm install morris.js06

Then add morris.min.js, morris.css and raphael-min.js to your page.

Raphaël can be added to your page through a CDN:

            
    <script src="//cdnjs.cloudflare.com/ajax/libs/raphael/2.1.2/raphael-min.js"></script>

With Angular / Ionic or Vue.js

Morris.js 06 can be easily used with these frameworks thanks to these 2 wrappers:

  • angular-morris-js
  • vue-morris

Your first chart

Start by adding a <div> to your page that will contain your chart. Make sure it has an ID so you can refer to it in your Javascript later.

            
    <div id="myfirstchart"></div>

Next add a <script> block to the end of your page, containing the following javascript code: 

            
    new Morris.Line({
      // ID of the element in which to draw the chart.
      element: 'myfirstchart',
      // Chart data records -- each entry in this array corresponds to a point on the chart.
      data: [
        { year: '2014', value: 20 },
        { year: '2015', value: 10 },
        { year: '2016', value: 5 },
        { year: '2017', value: 5 },
        { year: '2018', value: 20 }
      ],
      // The name of the data record attribute that contains x-values.
      xkey: 'year',
      // A list of names of data record attributes that contain y-values.
      ykeys: ['value'],
      // Labels for the ykeys -- will be displayed when you hover over the chart.
      labels: ['Value']
    });
